# Royal Vault

Royal Vault is a zero-knowledge password manager with a privacy-first architecture.

## Tech Stack
- Vue 3 + Vite + Tailwind
- Golang + Fiber
- PostgreSQL
- Client-side encryption

## Philosophy
Your secrets belong to you. Not the server.

## Project Status

Royal Vault is under active development. The repository will evolve to include:
- Core encryption and key management logic
- Secure authentication flows
- Scalable backend services
- A structured, user-friendly frontend

## Goal

To provide a trustworthy, transparent, and modern credential management solution that balances strong security guarantees with a simple user experience.
